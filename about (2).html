About me

I'm a front-end developer based in Lisbon who transforms ideas into functional web applications. I focus on writing clean, responsive code that creates engaging user interfaces across all devices.

User experience drives my development approach. I build with the end user in mind, ensuring every interaction feels intuitive and purposeful. Through projects ranging from weather applications to AI-powered tools, I've learned that great UX means balancing visual appeal with usability—from clear navigation patterns to accessible design choices. I believe great UX isn't just about making things look good; it's about understanding user needs and creating seamless experiences that solve real problems.

I'm exploring how AI can enhance web applications to create more personalized, adaptive experiences. My Portuguese Poem Generator and weather forecasting projects demonstrate how thoughtful AI integration can make applications more responsive to user needs while maintaining a human-centered approach. I see AI as a powerful tool for creating more responsive, adaptive interfaces that anticipate user needs.

Responsive design is fundamental to my work. Every project adapts seamlessly from mobile to desktop using HTML, CSS, JavaScript, Bootstrap, and Python. I ensure layouts remain functional and visually consistent regardless of screen size—essential in today's multi-device landscape.

I'm actively developing my skills through hands-on project work and engaging with Lisbon's tech community, always exploring the intersection of innovative technology and thoughtful design.
